### 字符串比较

[strcmp](https://www.tutorialspoint.com/c_standard_library/c_function_strcmp.htm)

例子：

```c
  ret = strcmp(str1, str2);

  if(ret < 0) {
    printf("str1 is less than str2");
  } else if(ret > 0) {
    printf("str2 is less than str1");
  } else {
    printf("str1 is equal to str2");
  }
```
